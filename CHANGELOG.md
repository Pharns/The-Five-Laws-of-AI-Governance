# Changelog — The Five Laws of AI Governance

All notable changes to the canonical doctrine are recorded here. This standard follows **semantic versioning** for a governance document:

- **MAJOR (vX.0)** — normative changes (the meaning or set of laws changes)
- **MINOR (v1.X)** — clarifications, added rationale, expanded crosswalks (no normative change)
- **PATCH (v1.0.X)** — typographical or formatting corrections only
